A MERN stack application implementing authentication, role-based access control,
and separate dashboards for Admin and Student users.

## Features
- JWT authentication
- Role-based access (Admin / Student)
- Admin CRUD operations on students
- Student profile view & update
- Protected routes

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB (Atlas)
- Auth: JWT, bcrypt

## Setup
1. Clone repo
2. Install deps for frontend & backend
3. Add .env in backend
4. Run both servers
